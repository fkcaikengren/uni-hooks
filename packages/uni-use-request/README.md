# @caikengren/uni-use-request

![TypeScript](https://img.shields.io/badge/TypeScript-5.x-blue?logo=typescript) ![Dependencies](https://img.shields.io/librariesio/release/npm/@caikengren/uni-use-request?label=dependencies) ![License](https://img.shields.io/npm/l/@caikengren/uni-use-request)

基于`vue-hooks-plus`的[useRequest](https://inhiblabcore.github.io/vue-hooks-plus/hooks/useRequest/quick-start)，做了小程序跨端处理，支持uni-app和h5。

## Install
- npm：`npm i @caikengren/uni-use-request`
- yarn：`yarn add @caikengren/uni-use-request`
- pnpm：`pnpm add @caikengren/uni-use-request`
